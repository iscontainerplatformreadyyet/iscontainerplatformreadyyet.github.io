www.iscontainerplatformreadyyet.com

Website about building container systems

The goal of listing all the categories of tools needed to build a container platform is to inform people, show the progress and what parts are still missing.

catagories:

  • logging
  • service discovery
  • clustering/leader election
  • container scheduler
  • machine scheduler

Because I've just started the website you can imagine pull requests and other helpful information would be very appreciated: https://github.com/iscontainerplatformreadyyet/iscontainerplatformreadyyet.github.io/
